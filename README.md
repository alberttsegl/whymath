# @alberttsegl/whymath

**Whymath** adalah library JavaScript murni yang dirancang untuk perhitungan finansial berbasis intent, bukan sekadar operator atau rumus. Alih-alih memaksa developer menghafal dan menulis formula kompleks, Whymath memungkinkan Anda menyatakan maksud bisnis secara natural, misalnya “hitung bunga tahunan dari pokok X dengan suku Y” dan library akan mengeksekusinya secara konsisten, deterministik, dan aman.

Dengan Whymath, developer bisa :
- Mengurangi human error karena salah pakai rumus atau operator
- Memastikan hasil selalu konsisten meski kode dijalankan di lingkungan berbeda
- Mudah diintegrasikan ke aplikasi web atau Node.js tanpa dependency berat

Whymath cocok untuk sistem finansial, e-commerce, payroll, accounting, dan semua aplikasi yang membutuhkan perhitungan yang reliable, akurat, dan mudah dibaca.

## Instalasi
```bash
npm i whymath
```
